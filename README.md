# Employee Attrition Analytics Platform

An end-to-end HR analytics and machine learning project for analyzing and predicting employee attrition.

This project was initially developed as a university team assignment and later independently extended into a full-scale analytics platform.

## Key Features
- Large-scale data processing using Polars LazyFrame (20M+ records)
- Deep exploratory data analysis (EDA)
- Automatic model selection (Logistic Regression vs XGBoost)
- Interactive executive dashboards using Streamlit and Plotly
- Employee retention risk assessment
- CV-based attrition analysis (PDF / TXT)

## Tech Stack
Python, Streamlit, Polars, Pandas, Plotly, XGBoost, Scikit-learn

## Notes
Dataset and trained models are excluded for size and privacy reasons.
